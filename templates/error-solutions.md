# Error Solutions

## How to Use
1. When error occurs, Claude checks this file first
2. If found → apply solution
3. If not found → solve and add entry

## Entry Format
```
### [Error message or type]
- **Context**: [when this occurs]
- **Cause**: [root cause]
- **Solution**: [how to fix]
- **Prevention**: [how to avoid]
- **Related files**: [affected files]
```
